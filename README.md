# TanStack Table React Demo

This is a minimal React app using Vite and TanStack Table v8 to display a sample table based on the Gettysburg Address data.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Files
- `src/App.jsx`: Main app component
- `src/TableDemo.jsx`: Table component using TanStack Table v8
- `package.json`: Project dependencies and scripts

## Data
The table displays sample data from the Gettysburg Address (word, length, index, count).
